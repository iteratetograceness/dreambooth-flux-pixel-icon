# icons-v3 — new training icons (round 3)

Drop new house-style icon PNGs here (any size; pipeline re-snaps the grid).
Name files descriptively (e.g. `umbrella-open.png`) — filenames seed the
captions. Covered geometry so far: thin appendages, tall verticals,
radiating lines, multi-object. Most wanted: long thin diagonals
(sword / bat / fishing rod / paintbrush) and tall striped verticals
(pencil / candle).
